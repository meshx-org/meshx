# controlplane-derive

Add `#[derive(CustomResource)]` to your custom resource struct.

## Installation

Add the `derive` feature to `kube`:

```toml
[dependencies]
kube = { version = "0.91.0", feature = ["derive"] }
```

## Usage

See the **[controlplane-derive API Docs](https://docs.rs/kube-derive/)**
